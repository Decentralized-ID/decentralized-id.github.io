    Ceramic is a decentralized data network that powers an ecosystem of interoperable Web3 applications and services. Ceramic‚Äôs event streaming protocol is a highly-scalable decentralized data infrastructure used for building all kinds of interoperable Web3 services and protocols, such as decentralized databases.     • Protocol Overview

      Ceramic is a decentralized event streaming protocol that enables developers to build decentralized databases, distributed compute pipelines, and authenticated data feeds, etc. Ceramic nodes can subscribe to subsets of streams forgoing the need of a global network state. This makes Ceramic an eventually consistent system (as opposed to strongly consistent like L1 blockchains), enabling web scale applications to be built reliably.

      The protocol doesn‚Äôt prescribe how to interpret events found within streams; this is left to the applications consuming the streams. One example of this type of application is ComposeDB.
